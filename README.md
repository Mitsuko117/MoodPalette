①課題名
MoodPalette

②課題内容（どんな作品か）
 ー カレンダーの日付をクリックすると、5種類の気分アイコンとコメント欄が表示されます。
 ー その日の気分と感じたことを簡単に記録できるアプリです。

③アプリのデプロイURL
https://mitsuko117.github.io/MoodPalette/

④アプリのログイン用IDまたはPassword（ある場合）
なし

⑤工夫した点・こだわった点
 ー 気分を“画像アイコン”として選択できるようにしたこと。
 ー 削除、保存、閉じるボタンをつけたこと。
 ー 選択中の気分が分かるよう、ホバー表示を加えたこと。

⑥難しかった点・次回トライしたいこと（又は機能）
 ー カレンダーの月を前後に移動できるようにすること（コードに一部コメントとして残っている状態）。
 ー 選択した気分やコメントがある日は、カレンダー上に✏アイコンを表示し、入力の有無が一目で分かるようにすること。
 　　（今は気分の数字のみ表示されている状態）
 ー 1ヶ月・1週間単位で気分の変化を可視化するグラフ機能の追加。
 ー AIを活用し、記録内容をもとにした簡易コメントを自動生成する機能の実装。
 
⑦フリー項目（感想、シェアしたいこと等なんでも）
  相変わらず難しい部分ばかりですが、ようやく少しずつ理解できる範囲が広がってきたと感じています。
  AIに頼りつつも、自分で書けるようになりたいので、忘れないうちに毎日少しでもコードに触れる時間を作っていこうと思います。
